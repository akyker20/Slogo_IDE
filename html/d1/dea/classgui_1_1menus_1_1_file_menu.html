Detailed Description

Class offers the user three major features. 1) The user can load an xml file to load a workspace (variables, screen parameters, pen parameters, etc.) 2) The user can save the workspace to an xml file for later retrieval 3) The user can create a new workspace - the user can do this in several different languages to allow globalization.

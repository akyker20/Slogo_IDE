42 In order to use this project, we require that you use an Ant build to make the Slogo.jar file that is used within the project.
43 
44 * What has to be done is right click the project slogo_team10, and click Properties
45 * The Builder section should be selected
46 * There should be an option to build a new Builder - make it an Ant Builder
47 * It can be titled MakeJar
48 * The build file should be set to "${workspace_loc:/slogo_team10/projectBuilder.xml}"
49 * The base directory should be set to "${workspace_loc:/slogo_team10}"
50 * After creating it, make sure the box next to the builder is checked
51 * The jar file that is required should now be automatically re-built anytime the program is run.
